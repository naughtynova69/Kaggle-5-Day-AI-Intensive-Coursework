
# Kaggle 5-Day Generative AI Coursework

This repository serves as a collection of resources and our group's contributions for completing the Kaggle 5-Day Generative AI coursework.

## Course Overview

The Kaggle 5-Day Generative AI course provides a hands-on introduction to the exciting field of generative artificial intelligence. It covers fundamental concepts, popular models, and practical applications through a series of daily lessons and exercises.

## Course Structure (as per Kaggle)

*   **Day 1: Introduction to Generative AI**
*   **Day 2: Large Language Models (LLMs)**
*   **Day 3: Applications of LLMs**
*   **Day 4: Introduction to Diffusion Models**
*   **Day 5: Advanced Generative AI Concepts**
